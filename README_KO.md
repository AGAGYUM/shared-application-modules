# 공통 애플리케이션 모듈

본 저장소는 사내 업무 시스템에서 공통으로 사용되던 JavaScript 및 Java 모듈을 정리한 공간입니다.
중복 코드를 줄이고, 일관된 UI/로직을 유지하기 위해 사용되었습니다.

## JavaScript 공통 모듈 (common.js)

### 1. 컴포넌트 처리
- 셀렉트 박스 초기화
- API 기반 옵션 로딩
- 드롭다운 필터 UI 처리
- 이벤트 재바인딩 처리

### 2. 문자열 처리
- HTML 태그 제거 및 텍스트 추출
- 이미지 경로 정규화 처리

### 3. 날짜 처리
- 오늘 날짜 포맷팅
- 월 시작일 / 말일 계산

### 4. 페이징 처리
- 페이지 그룹 단위 페이징
- 버튼 커스터마이징 지원
- 이벤트 중복 방지 구조
