# shared-application-modules

본 저장소는 실제 사내 업무 시스템에서 사용되던 공통 JavaScript 및 Java 모듈을 정리한 공간입니다.  
코드 재사용성 향상, 일관성 유지, 유지보수성 개선을 목적으로 구성되었습니다.

---

## JavaScript 공통 유틸리티

본 저장소에는 실제 사내 시스템에서 사용하던 JavaScript 공통 유틸리티 모듈이 포함되어 있습니다.  
여러 애플리케이션에서 반복적으로 사용되는 UI 동작을 표준화하고, 중복 코드를 줄이며, 전반적인 유지보수성을 높이기 위해 설계되었습니다.

### commonUtils.module.js

역할별 책임에 따라 구성된 재사용 가능한 JavaScript 유틸리티 모음입니다.

---

#### 1. 컴포넌트 처리 유틸리티
공통 UI 컴포넌트 처리를 위한 헬퍼 메서드를 제공합니다.

- 정적 옵션 목록 또는 API 응답 데이터를 기반으로 `<select>` 요소 초기화
- 기존 이벤트 제거 후 재바인딩을 통한 안전한 이벤트 처리
- 드롭다운 필터 UI 토글 및 활성 상태 관리

사용 예:
- 카테고리 셀렉트 박스 초기화
- 재사용 가능한 필터 드롭다운 컴포넌트
- 동적 DOM 환경에서의 이벤트 중복 방지 처리

---

#### 2. 문자열 처리 유틸리티
HTML 기반 문자열을 처리하기 위한 유틸리티 함수 모음입니다.

- HTML 문자열에서 텍스트만 추출
- `<img>` 태그 경로를 애플리케이션 컨텍스트에 맞게 정규화 및 변경

주요 활용 목적:
- 콘텐츠 정제(Sanitization)
- 환경별 HTML 렌더링 일관성 유지

---

#### 3. 날짜 처리 유틸리티
자주 사용되는 날짜 범위를 간편하게 처리하기 위한 경량 유틸리티입니다.

- 오늘 날짜를 `YYYY-MM-DD` 형식으로 반환
- 현재 월의 시작일 및 말일 계산

활용 예:
- 검색 조건 날짜 초기화
- 리포트 조회 기간 설정

---

#### 4. 페이징 처리 유틸리티
커스터마이징이 가능한 재사용형 페이징 컴포넌트입니다.

기능:
- 페이지 사이즈 및 페이지 그룹 크기 설정
- 페이지 버튼 및 네비게이션 화살표 커스터마이징 지원
- 내부 상태 관리 기반의 안전한 재렌더링 처리

사용 대상:
- 목록 페이지
- 데이터 기반 리포트 화면
- 동적 페이징이 필요한 대시보드 화면
